Roblet®-Server

Dieses Kapitel beschreibt, wie der Roblet®-Server, sein Werkzeug und die Bibliotheken eingesetzt werden können.  Mehr generelle Informationen bietet der Überblick für Server. Lesen Sie auch zunächst, was für generelle Voraussetzungen zu erfüllen sind, inklusive was für Anforderungen gestellt werden. 

In diesem Kapitel wird beschrieben, wie ein Server zum Laufen gebracht wird und bedient werden kann.  Durch Nutzung seines Werkzeugs kann eine zusätzliche Betreuung von Servern zur Laufzeit oder "post mortem" erreicht werden.  Das ist für Administratoren von Servern teilweise genauso interessant, wie für Entwickler von Software.  Und schließlich wird auch erläutert, wie man die Schnittstellen des Servers zur Entwicklung von Modulen benutzt.
