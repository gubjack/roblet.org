Interoperabilität

Auf dieser Seite ist aufgelistet, wie die einzelnen Klient- und Server-Versionen zusammenarbeiten.  Ein Server meldet bei Kontaktaufnahme durch einen Klienten immer alle ihm möglichen Versionen.  Der Klient wählt dann daraus die ihm erlaubte höchste Version aus.

  • 5 - Server kennt kein Download zum Klienten mehr
  • 4 - RMI-Marshalling eingestellt
  • 3 - Ferne Instanzen hinzugefügt
  • 2 - Ströme hinzugefügt
  • 1 - Verbindungen per TCP
  • RMI - Implementierung zum Finden einer geeigneten Schnittstelle
