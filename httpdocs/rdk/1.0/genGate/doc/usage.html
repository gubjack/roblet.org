Benutzung

Die Kombination genRob®-genGate/-genControl ist eine kommandozeilenorientierte Anwendung ohne grafische Oberfläche.

In diesem Kapitel finden Sie Details zur Funktionsweise, um ein Gefühl für den Betrieb zu bekommen.  Hier wird auch beschrieben, was für Voraussetzungen zu erfüllen sind, damit ein reibungslose Inbetriebnahme und störungsfreier Betrieb gewährleistet ist.

Weiterhin wird der Start der Anwendung mit möglichen bzw. teilweise notwendigen Parametern erläutert und beschrieben, wie man die Anwendung beendet.
