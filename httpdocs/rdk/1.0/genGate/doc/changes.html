Änderungen

Auf dieser Seite sind die Anpassungen, Erweiterungen und Änderungen aufgelistet, die für die vorliegende Version gemacht wurden.  Zuerst kommen die jüngsten (neuesten) Einträge.

Folgende Eintragsklassen gibt es:

  • Neue Dinge,
  • geänderte Sachen und
  • Fehlerkorrekturen.

Für Version 1.3

  • Umstellung auf genRob®-genControl 6.3 als Roblet®-Server, für leichtere Administration und höhere Sicherheit.

Für Version 1.2

  • Umstellung auf genRob®-genSite 3.3 für die Generierung der Dokumentation
  • Umstellung auf genRob®-genControl 6.1 als Roblet®-Server, um Protokoll 3 zur Verfügung zu haben.  Altes Klient-Server-Protokoll basierend auf RMI wird nicht mehr unterstützt.
  • Beispiel-Start-Script run nach run.sh umbenannt
  • Überarbeitung des Inhalts, damit wieder die Versionsnummern der abhängigen Komponenten leicht zu sehen sind
  • Leichte Vereinfachung der Einheiten-Javadoc - Einstiegsklasse des Java™-Archivs nun nicht mehr dokumentiert.

Für Version 1.1

  • Ä Anpassung von TCP- und Puffer-Parametern zur Optimierung der Kommunikationsgeschwindigkeit
  • N Ausgabe der gewählten Log-Schlüsselworte
  • Ä Überarbeitung der Start-Dateien zur besseren Nutzung
  • Ä Erweiterung der Beschreibung des Starts per Kommando

Für Version 1.0

  • N Initiale Version.
