genRob®-genGate 1.3 (18Apr10)

Willkommen!

genRob®-genGate wirkt als Stellvertreter für Roblet®-Server.  Dadurch können etwa nicht-lokale Server für einen Verzeichnisdienst in einem lokalen Netz sichtbar gemacht werden.  Das Paket ist als genRob®-genControl-Modul implementiert.  Eine passende Version von genRob®-genControl ist enthalten.

Ist Ihnen das Paket schon bekannt, so finden Sie unter Änderungen Informationen zu neuen oder angepassten Teilen.  Unter Inhalt sind Details zu den in diesem Paket mitgelieferten Dateien und Verzeichnissen zu finden.

Wollen Sie genRob®-genGate einrichten und mehr zum Umgang erfahren, so ist mehr unter Benutzung und Werkzeug zu finden.

Wollen Sie per Roblet® zugreifen, so müssen Sie Einheiten für Ihre Entwicklungen benutzen.

Weitere Hilfe

genRob®-genGate ist eine von mehreren Komponenten des genRob®-Systems der genRob® GmbH.  Mehr Informationen zur Roblet®-Technik sind bei roblet®.org zu finden.  Fragen und Hinweise sind sehr willkommen und können per e-Mail an genGate@genRob.com geschickt werden.
