Werkzeug

Mit dem Paket wird zusätzlich noch ein Werkzeug ausgeliefert.  Es unterstützt die Einrichtung und Wartung des Moduls.

Der Aufruf erfolgt in folgender Weise (in eine Zeile): 

java -jar genRob.genGate.tool.jar
	[Parameter]

Parameter

Die Parameterangabe [Parameter] folgt folgender Regel: 
host:port Einheit Befehl
host:port gibt den Roblet®-Server mit genRob®-genGate-Modul an.  Kombinationen von Einheit und Befehl können aus der weiter unten stehenden Tabelle entnommen werden.

Erfolgt der Aufruf ohne Parameter, so wird angezeigt, was mit dem Werkzeug alles gemacht werden kann.

Nachfolgend die Einheit-Befehl-Kombinationen und ihre Funktion:

Einheit Befehl Funktion
Version get Hole Versionsinformationen zum genGate-Modul vom angegebenen Server.
