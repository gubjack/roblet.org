Roblet®-Bibliothek 2.3 (10Jan10)

Dieses Paket "Roblet-Bibliothek" enthält die Klassen und Schnittstellen, die in einem verteilten System auf Basis der Roblet®-Technik in jeder der einzelnen Komponenten benötigt werden.  Man findet sie auf Anwendungs- wie auch Serverseite.

ACHTUNG:  Mit der Roblet®-Bibliothek allein entwickelt man keine Software.  Anwendungen und Server benötigen stattdessen weitere Software, über die die Kommunikation und das jeweilige Server-Verhalten festgelegt werden.  Für die Entwicklung von Roblet®-Anwendungen bzw. Server-Erweiterungen werden die Klienten-Bibliothek bzw. der Roblet®-Server genRob®-genControl benötigt, denen das Java™-Archiv der Roblet®-Bibliothek beiliegt.
Siehe roblet®.org für mehr Informationen.

Ist Ihnen die Roblet®-Bibliothek schon bekannt, so finden Sie unter Änderungen Informationen zu neuen oder angepaßten Teilen.  Wenn Ihnen die Bibliothek neu ist, verschaffen Sie sich am besten zunächst einen Überblick.

Unter Inhalt ist eine Auflistung aller Dateien dieses Paketes.  Benutzung erläutert, wie die Roblet®-Bibliothek eingesetzt werden kann.
