Benutzung

Das Kapitel Benutzung beschreibt, wie die in dem vorliegenden Paket enthaltenen Anwendungen und Bibliotheken eingesetzt werden können.

Dabei wird darauf eingegangen, was für generelle Voraussetzungen zu erfüllen sind, inklusive was für Anforderungen gestellt werden.  Es beschrieben, wie ein Server zum Laufen gebracht wird und bedient werden kann.  Durch Nutzung des Werkzeugs kann eine zusätzliche Betreuung von Servern zur Laufzeit oder "post mortem" erreicht werden.  Das ist für Administratoren von Servern teilweise genauso interessant, wie für Entwickler von Software.  Und schließlich wird auch erläutert, wie man die Schnittstellen des Servers zur Entwicklung von Modulen benutzt.
