Werkzeug

Das Werkzeug unterstützt die Wartung von Roblet®-Servern.  Es ist eine Java™-Anwendung ohne grafische Oberfläche.

Bei der Fernwartung (auch für lokal laufende Server) erfolgt der Aufruf in folgender Weise mit (in eine Zeile): 

java -jar genRob.genControl.tool.jar
    host:port Einheit Befehl

Zur Anzeige von Log-Dateien erfolgt der Aufruf in folgender Weise (in eine Zeile): 

java -jar genRob.genControl.tool.jar
    log  Datei1 Datei2@Versatz ...

Erfolgt der Aufruf ohne jeden Parameter, so wird angezeigt, was mit dem Werkzeug alles gemacht werden kann: 

java -jar genRob.genControl.tool.jar
