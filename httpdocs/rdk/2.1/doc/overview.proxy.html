Vertreterdienst

Ein Roblet®-Server-Vertreterdienst oder kürzer Vertreterdienst oder auch Vertreter wirkt als Stellvertreter für einen Roblet®-Server.  Dadurch können ferne Server, die auf Grund technischer Gegebenenheiten nicht direkt in sichtbaren Verzeichnisdiensten gelistet werden können, trotzdem eingetragen und damit in einem lokalen Netz sichtbar gemacht werden.  Damit können im lokalen Netz befindliche Anwendungen, den vertretenen Server während des laufenden Betriebes finden, ohne daß sie im voraus von ihm wissen müssen.
