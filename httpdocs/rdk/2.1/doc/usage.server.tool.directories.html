Verzeichnisdienste

Verzeichnisdienste lassen sich mit Hilfe des Werkzeugs verwalten.  Dabei ist nicht von Bedeutung, ob die jeweiligen Verzeichnisdienste auf dem gleichen Rechner laufen oder auf anderen.

Der Aufruf geschieht in folgender Weise (in eine Zeile): 

java  -jar org.roblet.jar  tool
    dir
    Befehl Parameter

Die möglichen Befehle können aus der nachfolgenden Tabelle entnommen werden.  Erfolgt der Aufruf ohne Befehl, so wird angezeigt, was mit dem Werkzeug im Zusammenhang mit Verzeichnisdiensten alles gemacht werden kann.

Befehl Parameter Funktion
list host:port Holt einmalig die Einträge vom Verzeichnisdienst mit dem Server-Namen host:port und gibt sie aus.
